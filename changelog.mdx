## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added advanced search with filters for content type, date range, and author
- Introduced collaborative editing with real-time cursors and conflict resolution
- Implemented automated documentation generation from OpenAPI specifications

## Improvements

- Enhanced performance for large documentation projects with lazy loading
- Improved mobile responsiveness across all documentation pages
- Updated editor with syntax highlighting for additional programming languages

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Complete redesign of the user interface with modern design system
- Added support for multiple documentation formats including MDX and plain markdown
- Introduced team workspaces with shared permissions and project isolation

## Breaking Changes

- API endpoints now require API key authentication for all requests
- Deprecated legacy webhook formats in favor of standardized payload structure
- Removed support for Internet Explorer browser

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Added version control integration with Git repositories
- Implemented webhook notifications for documentation changes
- Introduced template system for consistent documentation structure

## Bug Fixes

- Fixed issue with image uploads failing on certain file types
- Resolved search indexing problems with special characters
- Corrected permission inheritance in nested project structures

</Update>
